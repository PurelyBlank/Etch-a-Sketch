# Etch-a-Sketch

### Run Project
1) git clone the repository
    * `> git clone git@github.com:PurelyBlank/Etch-a-Sketch.git`
2) go into the respository
3) preview the index.html page on your web browser
    * i.e. `http://127.0.0.1:3000/index.html`
4) Viola!
